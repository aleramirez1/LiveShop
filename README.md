# LiveShop

Aplicación Android de marketplace con Clean Architecture + MVVM + Hilt.

## 🎯 Características

- Autenticación de usuarios (Login/Registro)
- Gestión de productos en tiempo real
- Búsqueda y filtrado de productos
- Interfaz moderna con Jetpack Compose
- Base de datos local con Room
- Inyección de dependencias con Hilt

## 🏗️ Arquitectura

- **Presentation Layer**: Composables + ViewModels
- **Domain Layer**: UseCases + Interfaces
- **Data Layer**: Repositorios + Room Database

## 📱 Requisitos

- Android 8.0 (API 26) o superior
- Android Studio Flamingo o superior
- Kotlin 1.9+

## 🚀 Instalación

```bash
git clone https://github.com/aleramirez1/LiveShop.git
cd LiveShop
```

Abre el proyecto en Android Studio y ejecuta en un emulador o dispositivo físico.
